Welcome to PyQvd's documentation!
=================================

**PyQvd** is a Python library that provides a simple API for reading/writing Qlik View Data (QVD) files
in Python. Using this library, it is possible to parse the binary QVD file format and convert it to a
Python object structure or vice versa.
